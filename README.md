# DotData

EZ Path is a website to aid immigrants understand simplified USCIS information.


Launched on a python server, uses HuggingFace API and GOOGLE Places API to access AI tools

Datasets obtained from USCIS Official site, curated and used to build regression model on XGBoost


AI Usage in project building:
  AI was used to help with debugging code, making suggestions, and explaining concepts. Helped with 
  understanding parsing data and building our XGBoost model as well connecting missing ends on the
  api implementations.
Internet Sources:
  HTML template from https://html5up.net/ 
  HuggingFace llama: https://huggingface.co/meta-llama/Llama-3.1-8B
  Google Places api: https://developers.google.com/maps/documentation/places/web-service/overview
  Datasets from: https://www.uscis.gov/tools/reports-and-studies/immigration-and-citizenship-data
